# 事故

这页是事故目录总览。

上游：[[projects/releases]]、[[projects/development/worklog]]、[[projects/decisions]]、[[projects/memory/README]]  \
下游：每一条独立事故记录

## 这页负责什么

- 汇总当前项目的事故与复盘记录
- 说明事故相关的整体状态
- 链接到每一条独立事故记录
- 保留跨事故的共性结论和改进项

## 使用方式

- 每一个事故单独建一个文件
- 文件名尽量稳定，建议带日期和主题
- 这页只做总览、状态和入口，不把所有事故细节堆在这里

示例命名：

- `2026-04-09-login-timeout.md`
- `2026-04-12-payment-retry-loop.md`

## 当前内容

按需要维护：

- 当前事故状态总览
- 事故索引
- 共性问题
- 后续改进项

## 维护说明

- 如果事故结论会长期影响项目推进，回写到 `projects/memory/README.md`、`POLICY.md` 或 `projects/decisions.md`
- 事故目录只做索引和复盘入口，不重复堆正文
