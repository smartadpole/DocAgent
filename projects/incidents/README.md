# 事故

这页是事故目录总览。

上游：[[projects/releases]]、[[projects/development/worklog]]、[[projects/decisions]]

## 这页负责什么

- 汇总当前项目的事故与复盘记录
- 说明事故相关的整体状态
- 链接到每一条独立事故记录
- 保留跨事故的共性结论和改进项

## 使用方式

- 每一个事故单独建一个文件
- 文件名尽量稳定，建议带日期和主题
- 这页只做总览、状态和入口，不把所有事故细节堆在这里

示例命名：

- `2026-04-09-login-timeout.md`
- `2026-04-12-payment-retry-loop.md`
