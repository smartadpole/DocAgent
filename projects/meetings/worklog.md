# 会议记录

主入口：[[projects/meetings/README]]

这页记录正式会议的时间线。

如果会议很多，就让这页专门承接“会议发生了什么、结论是什么、要分流到哪”。
如果只是开发过程里的临时讨论，优先写到 [[projects/development/worklog]]。
默认按时间倒序维护，新的会议记录放前面。

## 这页负责什么

- 记录正式会议的过程和结果
- 记录会后行动项和责任人
- 记录需要回到决策、需求或开发页的内容
- 作为会议材料的时间线索引

## 这页不负责什么

- 不做逐字转录
- 不保存原始录音、截图或下载附件本体
- 不替代 [[projects/decisions]]、[[projects/trace]] 或 [[projects/development/worklog]]

## 记录模板

每场会议尽量按这个骨架写：

```md
- 时间：
- 会议名称：
- 议题：
- 参会人：
- 结论：
- 行动项：
- 分流：
- 关联链接：
```

## 记录方式

- 一场正式会议对应一条记录
- 同一场会如果有后续补充，优先补到原记录下面，不另起分散条目
- 会议结果如果已经稳定，及时回写到对应主页面，不只留在这里
