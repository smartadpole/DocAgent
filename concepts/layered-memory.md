# 分层 Memory

## 定义

分层 memory 是一种把临时信息、稳定偏好、项目共识和决策规则分开存放，并按优先级注入后续任务的上下文系统设计方法。

它关注的重点不是“记住多少内容”，而是：

- 什么该记
- 记到哪一层
- 什么时候生效
- 冲突时怎么办
- 何时升级为规则或资产
